# 移动端设计规范 作业

找一个知名产品（有相关工作经验的学员可以选择自己参与的产品）出一份移动端设计规范
要求：

- 1.表达清晰，内容详细完整；
- 2.字体规范，包含app内文字根据使用场景的归类；
- 3.颜色规范，主色、辅色、背景色等颜色的说明和具体色值标注；
- 4.控件规范，每个具体控件的尺寸大，及布局方式的说明；
- 5.标注清晰正确且具体，可将一些说明性文字写在左边区域
